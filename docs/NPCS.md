# Planet Miner — NPCs

## Hub-NPCs (Stand: designed, noch nicht implementiert)

### Wizzle the Wizard
- **Rolle:** Dimensionsmaster — Stage-/Portal-Wechsel
- **Standort:** Hub (Raumschiff)
- **Funktion:** Spieler wählt über Wizzle den nächsten Planeten/Stage
- **Tickets:** PM-041
- **Gameplay:** Ersetzt das bisherige Dimension Portal UI

### Chubbs the Trader
- **Rolle:** Händler — Materialien verkaufen
- **Standort:** Hub (Raumschiff)
- **Funktion:** Spieler verkauft gesammelte Materialien gegen Währung
- **Tickets:** PM-042
- **Gameplay:** Ersetzt das bisherige Sell-Board

### Drillo the Miner
- **Rolle:** Premium Gear Seller — Robux-Pickaxe-Shop
- **Standort:** Hub (Raumschiff)
- **Funktion:** Spezial-Pickaxes für Robux kaufen
- **Tickets:** PM-043
- **Gameplay:** Premium-Monetarisierung, kosmetische + stärkere Pickaxes

### Tinker the Upgrader
- **Rolle:** Ingenieur — Damage/Speed/Luck Upgrades
- **Standort:** Hub (Raumschiff)
- **Funktion:** Spieler investiert Währung in permanente Upgrades
- **Tickets:** PM-044
- **Gameplay:** Haupt-Progression neben Pickaxes
